# Alimentação saudável com baixo custo

Este projeto é uma interface web desenvolvida com HTML, CSS e JS com o objetivo de apresentar o tema sobre alimentação saudável e acessível.

## Objetivo
Criar uma página visualmente agradável e organizada, focada em transmitir informações de forma simples e clara, utilizando design leve e cores que remetem à saúde e bem-estar.

## Tecnologias utilizadas
- HTML
- CSS
- JS

## Design
O layout foi inteiramente pensado com uma tabela de cores baseada em azul, bege, vermelho e preto, conforme a logo. Busca transmitir tranquilidade, saúde e equilíbrio.

## Responsividade
A interface foi construída com foco em adaptação para diferentes tamanhos de tela.
